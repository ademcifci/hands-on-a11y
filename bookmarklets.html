Accessibility Bookmarklets

Use the bookmarklets below to help with Accessibility testing

  • Language Checker Bookmarklet (doesn't check for 'xml:lang')
  • Force Visual Focus Bookmarklet
  • Check for tabindex attribute
  • 200% Text Resize (Note: only works in IE)
  • Add focusable=false to SVGs (adds focusable="false" to all svg elements on a page)
  • Find aria-* attributes
  • Page Title
  • List Links
  • List Headings (shows all headings, even display none)
  • Visible Headings (source - http://developer-guide.access4all.ch/en/pages/visible_heading_bookmarklet shows only visible headings)
  • List Images
  • Show Images (source - http://pauljadam.com/bookmarklets/)
  • Lists (source - http://pauljadam.com/bookmarklets/)
  • Disable CSS
  • Form Labels (source - http://jimthatcher.com/favelets/)
  • Skip Links (source - http://jimthatcher.com/favelets/)
  • Data Tables (source - http://jimthatcher.com/favelets/)
